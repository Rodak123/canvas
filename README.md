# Canvas

A r/place inspired website.

## Setup

### Server

1. Create a `.env` from `.env.example`
2. Install modules 
```sh
npm install
```
3. Start server
```sh
npm run start
```

### Client

Inside `/canvas-client`

1. Create a `.env` from `.env.example`
2. Install modules
```sh
npm install
```
3. Start server
```sh
npm run dev
```
